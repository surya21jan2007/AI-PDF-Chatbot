# 📚 AI PDF Chatbot

An AI-powered PDF Question Answering application built using Streamlit, LangChain, FAISS, HuggingFace Embeddings, and GPT4All.

Users can upload a PDF document and ask questions in natural language. The chatbot retrieves relevant information from the document and generates accurate answers using Retrieval-Augmented Generation (RAG).

---

## 🚀 Features

* Upload PDF documents
* Automatic text chunking
* Semantic search using embeddings
* FAISS vector database
* Question answering from document content
* Source page references
* Simple and user-friendly Streamlit interface

---

## 🛠️ Tech Stack

* Python
* Streamlit
* LangChain
* HuggingFace Embeddings
* FAISS
* GPT4All
* PyPDF

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/AI-PDF-Chatbot.git
cd AI-PDF-Chatbot
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
